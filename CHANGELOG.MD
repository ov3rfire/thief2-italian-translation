# Changelog

## 0.2.0 - Ominous Bequest Gold e aggiornamento traduzioni

Aggiornamento della traduzione italiana non ufficiale per **Thief II: The Metal Age**, con aggiunta di una nuova Fan Mission tradotta.

### Aggiunto

- Aggiunta la traduzione italiana della Fan Mission `fm_ominousbequestgold`.
- Aggiunta la Fan Mission **Ominous Bequest Gold** alla struttura del repository.
- Aggiunte nel README indicazioni per chi vuole proporre Fan Mission da tradurre.
- Aggiunte nel README indicazioni per chi possiede sottotitoli o file testuali di Fan Mission in altre lingue e vuole metterli a disposizione per una possibile traduzione italiana.

### Aggiornato

- Aggiornate alcune traduzioni del gioco originale **Thief II: The Metal Age**.
- Aggiornate alcune traduzioni della Fan Mission `fm_brokentriad1_1_0`.
- Aggiornato il README con la nuova Fan Mission disponibile.
- Aggiornate le istruzioni relative all’installazione delle Fan Mission.
- Aggiornato il pacchetto consigliato alla versione `v0.2.0`.

### Pacchetti disponibili

- `Thief2-Italian-Translation-Pack-v0.2.0.zip`  
  Pacchetto completo con gioco originale e tutte le Fan Mission attualmente disponibili.

- `Ominous-Bequest-Gold-Italian-v0.2.0.zip`  
  Pacchetto dedicato solo alla Fan Mission **Ominous Bequest Gold**.

### Note

- La versione consigliata rimane `con_apostrofi`.
- La traduzione continua a essere distribuita come file sciolti compatibili con NewDark/T2Fix.
- I file `.crf` originali non vengono modificati.
- È consigliato fare un backup della cartella `MODS\Subtitles\SUBTITLES` prima di installare i sottotitoli tradotti.

## 0.1.0 - Prima versione pubblica testabile

Prima versione pubblica testabile della traduzione italiana non ufficiale per **Thief II: The Metal Age**.

### Aggiunto

- Traduzione italiana realizzata da zero per il gioco originale.
- Traduzione di menu e interfaccia.
- Traduzione degli obiettivi missione.
- Traduzione di libri, pergamene e testi leggibili.
- Traduzione di nomi oggetti e descrizioni.
- Traduzione dei testi di gioco presenti nei file traducibili.
- Traduzione dei sottotitoli inclusi in T2Fix/NewDark, dove disponibili.
- Prima struttura per traduzioni italiane di Fan Mission.
- Prima Fan Mission inclusa: `fm_brokentriad1_1_0`.
- Distribuzione come file sciolti compatibili con NewDark.
- Versioni alternative della traduzione:
  - `con_accenti`
  - `senza_accenti`
  - `con_apostrofi`

### Compatibilità testata

La traduzione è stata realizzata e testata su:

- `setup_thief_2_-_the_metal_age_1.26_nd_(21946)`
- `T2Fix_1.27-2025-08-10_with_mods`

### Note

- I file `.crf` originali non vengono modificati.
- La versione consigliata è `con_apostrofi`, perché alcune schermate di Thief II/NewDark non visualizzano correttamente le lettere accentate italiane.
- È consigliato fare un backup della cartella `MODS\Subtitles\SUBTITLES` prima di installare i sottotitoli tradotti.
