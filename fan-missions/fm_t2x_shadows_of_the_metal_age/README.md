# T2X: Shadows of the Metal Age - Traduzione italiana

Traduzione italiana non ufficiale per la Fan Mission **T2X: Shadows of the Metal Age** per **Thief II: The Metal Age**.

Questa traduzione è distribuita come pacchetto separato dedicato solo a **T2X: Shadows of the Metal Age**.

## Versione

```text
v0.2.1
```

## File release consigliato

```text
T2X-Shadows-of-the-Metal-Age-Italian-v0.2.1.zip
```

Gli archivi automatici generati da GitHub, come `Source code (zip)` e `Source code (tar.gz)`, non sono il pacchetto consigliato per l’utente finale.

## Versione consigliata

La versione consigliata è:

```text
con_apostrofi
```

Questa versione sostituisce le lettere accentate con apostrofi, per evitare problemi di visualizzazione in alcune schermate di Thief II/NewDark.

Esempio:

```text
È perché può così
```

diventa:

```text
E' perche' puo' cosi'
```

## Contenuto del pacchetto

Il pacchetto contiene:

README.md

italian/con_apostrofi/loose_files/books/

La traduzione di T2X si trova in:

```text
italian/con_apostrofi/loose_files/
```

## Nota importante sulla struttura di T2X

**T2X: Shadows of the Metal Age** usa una struttura diversa da alcune altre Fan Mission.

In questa traduzione non è presente la cartella lingua `italian` dentro `books`.

La struttura corretta è:

```text
italian/con_apostrofi/loose_files/books/
```

e non:

```text
italian/con_apostrofi/loose_files/books/italian/
```

Questo significa che i file tradotti devono essere copiati direttamente sopra i file originali della Fan Mission, mantenendo la struttura presente dentro `loose_files`.

## Installazione

1. Fare un backup completo della cartella della Fan Mission **T2X: Shadows of the Metal Age**.

2. Aprire la cartella:

```text
italian/con_apostrofi/loose_files/
```

3. Copiare tutto il contenuto di `loose_files` dentro la cartella della Fan Mission T2X installata.

Esempio:

```text
C:\Giochi\Thief 2\Thief 2\FMs\T2X\
```

Il nome effettivo della cartella può variare in base a come la Fan Mission è stata installata o gestita dal proprio loader.

4. Dopo la copia, la struttura nella cartella della Fan Mission dovrebbe includere percorsi simili a:

```text
C:\Giochi\Thief 2\Thief 2\FMs\T2X\books\
```

5. I file tradotti devono sostituire direttamente i file originali della Fan Mission.

## Configurazione lingua

Per T2X questa traduzione sostituisce direttamente i file testuali originali presenti nella struttura della Fan Mission.

Non è necessario creare una cartella `books\italian`.

Se la Fan Mission usa un file `cam_mod.ini`, non modificare la struttura dei file tradotti: seguire sempre la struttura presente dentro `loose_files`.

## Backup consigliato

Prima di installare la traduzione è fortemente consigliato fare un backup completo della cartella della Fan Mission.

Esempio:

```text
C:\Giochi\Thief 2\Thief 2\FMs\T2X\
```

In caso di problemi, sarà possibile ripristinare i file originali della Fan Mission.

## Disinstallazione

Per rimuovere la traduzione, ripristinare il backup della cartella della Fan Mission.

In alternativa, reinstallare la Fan Mission originale o ripristinare manualmente i file originali sostituiti.

## Note tecniche

La traduzione è distribuita come file sciolti.

Per T2X i file tradotti devono seguire la struttura originale della Fan Mission e sostituire direttamente i file testuali corrispondenti.

T2X può usare missioni e cartelle con numerazione non standard, per esempio `miss19`, `miss20`, `miss21` e simili.

Non bisogna quindi dare per scontato che la missione 3 corrisponda necessariamente a `miss3`.
